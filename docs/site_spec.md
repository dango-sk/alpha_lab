# Alpha Lab — 퀀트 전략 대시보드 기획서

> 이현자산운용 퀀트팀 내부용 대시보드. 멀티팩터 전략의 백테스트 결과를 시각화하고, 포트폴리오를 분석하며, AI 챗을 통해 데이터를 탐색하는 도구.

---

## 1. 퍼포먼스 (`/performance`)

**용도**: 전략별 성과 비교 — 메인 화면

| 영역 | 설명 |
|------|------|
| **필터 바** | 유니버스(KOSPI / KOSPI+KOSDAQ), 리밸런싱 주기(월간 / 격주), 날짜 범위 선택 |
| **KPI 카드** | 전략별 누적 수익률 · MDD · Sharpe |
| **성과 요약 테이블** | CAGR, MDD, Sharpe, 평균 턴오버, 평균 종목 수 |
| **누적 수익률 차트** | 포트폴리오 가치 성장 곡선 (IS/OOS 구간 표시) |
| **드로다운 차트** | 최대 낙폭 추이 |
| **연도별 월간 수익률** | 캘린더 히트맵 (연도 × 12개월 + YTD) + 연간 통계 |
| **IS/OOS 비교** | 인샘플 vs 아웃오브샘플 수익률·Sharpe |
| **롤링 12개월 초과수익률** | 벤치마크 대비 초과수익 추이 |
| **레짐 분석** | 상승/횡보/하락장 구간 분류 + 구간별 전략 성과 테이블 + 평균 월간 수익률 바 차트 |

---

## 2. 포트폴리오 (`/portfolio`)

**용도**: 특정 리밸런싱 시점의 보유 종목·섹터·시총 분석

| 영역 | 설명 |
|------|------|
| **날짜 선택** | 리밸런싱 날짜 드롭다운 |
| **전략 선택** | 복수 전략 체크 비교 |
| **포트폴리오 특성** | 가중평균 PER, PBR, EV/EBITDA |
| **집중도 분석** | HHI, Top 5 비중, 종목 수 |
| **Top 5 보유 비교** | 전략 간 상위 5개 종목 스캐터 |
| **섹터 배분** | 가로 바 차트 (전략별 섹터 비중) |
| **시총 분포** | 초대형/대형/중형/소형 분류 바 차트 + 요약 테이블 |
| **리밸런싱 턴오버** | 신규 편입·제외·유지 종목 수 + 턴오버율 + 상세 테이블 |
| **보유 종목 상세** | 전략별 전 종목 리스트 (점수, PER, PBR, EV/EBITDA, 월간 수익률) |

---

## 3. 월간 분석 (`/monthly`)

**용도**: 월별 수익률 분해 및 종목 기여도(어트리뷰션) 분석

| 영역 | 설명 |
|------|------|
| **월간 수익률 히트맵** | 전략 × 월 셀, 색상으로 +/− 표시. 클릭하면 드릴다운 |
| **드릴다운 어트리뷰션** | Top 5 / Bottom 5 기여 바 차트 + 전 종목 기여도 테이블 (종목명, 섹터, 비중, 수익률, 기여도) |
| **롤링 12개월 초과수익률** | KOSPI 대비 12개월 누적 초과수익 추이 |

---

## 4. 통계 검증 (`/statistics`)

**용도**: 전략 성과의 통계적 유의성 검증

| 영역 | 설명 |
|------|------|
| **IS vs OOS** | 인샘플/아웃오브샘플 Sharpe 비교 바 차트 + 상세 테이블 |
| **부트스트랩 유의성** | t-통계량 바 차트 (±1.96 기준선) + 평균 월간 초과수익률, p-value, 95% CI, 승률 |
| **롤링 24개월 윈도우** | 24개월 롤링 초과수익 차트 + 양의 구간 비율 (win rate) |

---

## 5. 실험실 (`/lab`)

**용도**: 커스텀 전략 생성·편집·백테스트

| 영역 | 설명 |
|------|------|
| **저장 전략 로드** | 이전에 저장한 전략 불러오기/삭제 |
| **전략 설정** | 유니버스, 비중 상한(%), Top N 종목 수, 거래비용(bp), 리밸런싱 주기 |
| **팩터 가중치 에디터** | 카테고리별(밸류에이션·회귀매력도·성장성·차별화) 슬라이더로 개별 팩터 비중 조절 |
| **전략 코드 뷰어** | 생성된 Python 전략 코드 확인/수정 |
| **백테스트 실행** | 진행률 바 + 결과 비교 테이블 (기존 vs 수정 vs 벤치마크) + 누적 수익률 차트 |
| **저장** | 전략명·설명 입력 후 저장 |

## 6. AI 챗 (`/chat`)

**용도**: 자연어로 전략 분석·데이터 조회·전략 추천

| 영역 | 설명 |
|------|------|
| **세션 사이드바** | 대화 이력 관리 (신규 생성, 목록, 삭제) |
| **채팅 영역** | 마크다운 렌더링 (표, 코드, 차트), SQL 실행 결과 인라인 표시 |
| **전략 추천** | AI가 Python 전략 코드를 제안하면 "실험실에서 열기" 버튼 표시 |
| **추천 프롬프트** | 새 세션 시 분석 예시 질문 표시 |

## 데이터 흐름

```
PostgreSQL (일별 시세·재무·컨센서스)
    ↓
FastAPI 백엔드 (/api/results, /api/holdings, /api/attribution, ...)
    ↓
Next.js 프론트엔드 (fetch → state → Plotly 차트 + 테이블)
```

## 전략 팩터 구조

| 카테고리 | 팩터 | 의미 |
|----------|------|------|
| **밸류에이션** | T_PER, F_PER, T_PBR, F_PBR, T_EVEBITDA, F_EVEBITDA, T_PCF | 12개월 트레일링(T) / 포워드(F) 밸류에이션 지표의 매력도 |
| **회귀 매력도** | ATT_PBR, ATT_EVIC, ATT_PER, ATT_EVEBIT | ROE-PBR, ROIC-EV/IC 등 회귀선 대비 괴리율 (양수 = 저평가) |
| **성장성** | T_SPSG, F_SPSG | 매출 성장률 (트레일링/포워드) |
| **차별화** | F_EPS_M | 컨센서스 EPS 변화율 (모멘텀) |

각 팩터에 0~100% 가중치를 부여하여 종합 점수 산출 → 상위 N개 종목 선정 → 시총 비례 비중 배분 (개별 상한 10%)
