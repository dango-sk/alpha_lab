01

퍼포먼스 /performance

전략별 성과 비교 — 메인 화면

⚙ 필터 바
유니버스(KOSPI / KOSPI+KOSDAQ), 리밸런싱 주기(월간 / 격주), 날짜 범위 선택
◼ KPI 카드
전략별 누적 수익률, MDD, Sharpe
◼ 성과 요약 테이블
CAGR, MDD, Sharpe, 평균 턴오버, 평균 종목 수
◼ 누적 수익률 차트
포트폴리오 가치 성장 곡선, IS/OOS 구간 표시
◼ 드로다운 차트
최대 낙폭 추이
◼ 연도별 월간 수익률
캘린더 히트맵 (연도 × 12개월 + YTD) + 연간 통계
◼ IS/OOS 비교
인샘플 vs 아웃오브샘플 수익률, Sharpe
◼ 롤링 12개월 초과수익률
벤치마크 대비 초과수익 추이
◼ 레짐 분석
상승/횡보/하락장 구간 분류 + 구간별 전략 성과 + 평균 월간 수익률
02

포트폴리오 /portfolio

특정 리밸런싱 시점의 보유 종목, 섹터, 시총 분석

◼ 날짜 / 전략 선택
리밸런싱 날짜 드롭다운 + 복수 전략 체크 비교
◼ 포트폴리오 특성
가중평균 PER, PBR, EV/EBITDA
◼ 집중도 분석
HHI, Top 5 비중, 종목 수
◼ Top 5 보유 비교
전략 간 상위 5개 종목 스캐터
◼ 섹터 배분
가로 바 차트 — 전략별 섹터 비중
◼ 시총 분포
초대형/대형/중형/소형 분류 + 요약 테이블
◼ 리밸런싱 턴오버
신규 편입/제외/유지 종목 수, 턴오버율, 상세 테이블
◼ 보유 종목 상세
전 종목 리스트 — 점수, PER, PBR, EV/EBITDA, 월간 수익률
03

통계 검증 /statistics

전략 성과의 통계적 유의성 검증

◼ IS vs OOS
인샘플/아웃오브샘플 Sharpe 비교 바 차트 + 상세 테이블
◼ 부트스트랩 유의성
t-통계량 (±1.96 기준선), p-value, 95% CI, 승률
◼ 롤링 24개월 윈도우
24개월 롤링 초과수익 차트 + 양의 구간 비율 (win rate)
04

실험실 /lab

커스텀 전략 생성, 편집, 백테스트

◼ 전략 관리
저장된 전략 불러오기/삭제
◼ 전략 설정
유니버스, 비중 상한(%), Top N, 거래비용(bp), 리밸런싱 주기
◼ 팩터 가중치 에디터
카테고리별 슬라이더로 개별 팩터 비중 조절 (밸류에이션, 회귀매력도, 성장성, 차별화)
◼ 백테스트 실행
진행률 바 + 기존 vs 수정 vs 벤치마크 비교 테이블 + 누적 수익률 차트
05

AI 챗 /chat

전략 팩터 구조

밸류에이션 — 트레일링(T) / 포워드(F) 밸류에이션 매력도
T_PER F_PER T_PBR F_PBR T_EVEBITDA F_EVEBITDA T_PCF
회귀 매력도 — 회귀선 대비 괴리율 (양수 = 저평가)
ATT_PBR ATT_EVIC ATT_PER ATT_EVEBIT
성장성 — 매출 성장률
T_SPSG F_SPSG
차별화 — 컨센서스 EPS 변화율 (모멘텀)
F_EPS_M
각 팩터에 0~100% 가중치를 부여하여 종합 점수 산출 → 상위 N개 종목 선정 → 시총 비례 비중 배분 (개별 상한 10%)
