1. 퍼포먼스 /performance

전략별 성과 비교 — 메인 화면

영역설명
필터 바유니버스(KOSPI / KOSPI+KOSDAQ), 리밸런싱 주기(월간 / 격주), 날짜 범위 선택
KPI 카드전략별 누적 수익률, MDD, Sharpe
성과 요약 테이블CAGR, MDD, Sharpe, 평균 턴오버, 평균 종목 수
누적 수익률 차트포트폴리오 가치 성장 곡선 (IS/OOS 구간 표시)
드로다운 차트최대 낙폭 추이
연도별 월간 수익률캘린더 히트맵 (연도 × 12개월 + YTD) + 연간 통계
IS/OOS 비교인샘플 vs 아웃오브샘플 수익률, Sharpe
롤링 12개월 초과수익률벤치마크 대비 초과수익 추이
레짐 분석상승/횡보/하락장 구간 분류 + 구간별 전략 성과 테이블 + 평균 월간 수익률 바 차트

2. 포트폴리오 /portfolio

특정 리밸런싱 시점의 보유 종목, 섹터, 시총 분석

영역설명
날짜 선택리밸런싱 날짜 드롭다운
전략 선택복수 전략 체크 비교
포트폴리오 특성가중평균 PER, PBR, EV/EBITDA
집중도 분석HHI, Top 5 비중, 종목 수
Top 5 보유 비교전략 간 상위 5개 종목 스캐터
섹터 배분가로 바 차트 (전략별 섹터 비중)
시총 분포초대형/대형/중형/소형 분류 바 차트 + 요약 테이블
리밸런싱 턴오버신규 편입/제외/유지 종목 수 + 턴오버율 + 상세 테이블
보유 종목 상세전략별 전 종목 리스트 (점수, PER, PBR, EV/EBITDA, 월간 수익률)

3. 통계 검증 /statistics

전략 성과의 통계적 유의성 검증

영역설명
IS vs OOS인샘플/아웃오브샘플 Sharpe 비교 바 차트 + 상세 테이블
부트스트랩 유의성t-통계량 바 차트 (±1.96 기준선) + 평균 월간 초과수익률, p-value, 95% CI, 승률
롤링 24개월 윈도우24개월 롤링 초과수익 차트 + 양의 구간 비율 (win rate)

4. 실험실 /lab

커스텀 전략 생성, 편집, 백테스트

영역설명
저장 전략 로드이전에 저장한 전략 불러오기/삭제
전략 설정유니버스, 비중 상한(%), Top N 종목 수, 거래비용(bp), 리밸런싱 주기
팩터 가중치 에디터카테고리별(밸류에이션, 회귀매력도, 성장성, 차별화) 슬라이더로 개별 팩터 비중 조절
전략 코드 뷰어생성된 Python 전략 코드 확인/수정
백테스트 실행진행률 바 + 결과 비교 테이블 (기존 vs 수정 vs 벤치마크) + 누적 수익률 차트
저장전략명, 설명 입력 후 저장

전략 팩터 구조

카테고리팩터의미
밸류에이션 T_PERF_PERT_PBRF_PBRT_EVEBITDAF_EVEBITDAT_PCF 12개월 트레일링(T) / 포워드(F) 밸류에이션 지표의 매력도
회귀 매력도 ATT_PBRATT_EVICATT_PERATT_EVEBIT ROE-PBR, ROIC-EV/IC 등 회귀선 대비 괴리율 (양수 = 저평가)
성장성 T_SPSGF_SPSG 매출 성장률 (트레일링 / 포워드)
차별화 F_EPS_M 컨센서스 EPS 변화율 (모멘텀)
각 팩터에 0~100% 가중치를 부여하여 종합 점수 산출 → 상위 N개 종목 선정 → 시총 비례 비중 배분 (개별 상한 10%)
